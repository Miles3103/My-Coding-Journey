=== LEVEL 14 - VARIABLES [3/3] : Swap Without Temp ===
FILE: rendu/lvl14/task.c

Swap two integers WITHOUT a temporary variable.
Use XOR swap or arithmetic swap.

Start: a = 5, b = 9

Expected output:
  Before: a=5, b=9
  After:  a=9, b=5

Tip XOR swap: a ^= b; b ^= a; a ^= b;
Tip arith:    a = a+b; b = a-b; a = a-b;
