=== LEVEL 57 - POINTERS [1/3] : Pointer Arithmetic ===
FILE: rendu/lvl57/task.c

int arr[] = {10, 20, 30, 40, 50};
int *p = arr;

Navigate using ONLY pointer arithmetic (no arr[i]):
  *p     = 10
  *(p+1) = 20
  *(p+2) = 30
  *(p+3) = 40
  *(p+4) = 50

Also print:
  int units apart:  1
  bytes apart:      4

Tip: (char*)&arr[1] - (char*)&arr[0] gives bytes.
     &arr[1] - &arr[0] gives int units.
