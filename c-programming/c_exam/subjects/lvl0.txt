Write a complete C program that prints:
  Hello, C World!
using write() ONLY. No printf allowed.

Rules:

  - Use write(1, ..., len)
  - End with newline

Expected output:
  Hello, C World!

Tip: write(1, "Hello, C World!\n", 16);
